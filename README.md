# 沖繩親子之旅 · 行程規劃

互動式沖繩一週親子旅行規劃網站，專為夫婦二人帶 1 歲半幼兒的 2026 年 9–10 月行程設計。

## 功能

- **總覽** — 9–10 月天氣、行前速查、帶幼兒旅行小撇步
- **行程規劃** — 7 日可編輯行程，支援排序、備忘錄，自動儲存
- **景點探索** — 20+ 精選景點，可依類別篩選、搜尋，一鍵加入行程
- **行李清單** — 分類 packing list，含寶寶用品與證件
- **預算估算** — 可調整各項費用，即時計算總預算

## 開始使用

```bash
npm install
npm run dev
```

瀏覽器開啟 http://localhost:5173

## 建置

```bash
npm run build
npm run preview
```

## 技術

- React 19 + TypeScript + Vite
- CSS Modules
- localStorage 持久化（行程、清單、預算）
